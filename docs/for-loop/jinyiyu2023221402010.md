## 问题描述 ##
通过编写计算机脚本，从1加到100是一个可以利用for循环来轻松解决的问题，如下给出了利用c语言实现的代码。
```
#include<stdio.h>
int main()
{
	int i, sum = 0;
	for(i = 1;i <= 100;i++)
		sum += i;
	printf("%d\n", sum);
}
```
## 代码拆解 ##
- ### 预处理 ###
  - 程序在编译之前需要的预处理预操作，
  - `#include<stdio.h>`主要用来提供所需的标准库，其中`stdio.h`就是标准输入输出头文件，包含了`main`函数中所用到的`printf`函数。
- ### 程序主体 ###
  - 程序一般从``main``函数开始。函数主体用``{}``包起来。
  - 首先：对需要用到的变量进行初始化，这里的``i``表示``1，2，3，...，100``。``sum``则表示计算结果。``int``表示指定该对象为整型。
  - 其次：`for`循环的第一个分号之前`i=1`，表示循环从1开始，接着判断`i`是否小于等于`100`，
    - 如果i小于等于`100`，则执行循环主体，即`sum+=i`;
      - 然后，执行`i++`，表示`i`自增`1`;
      - 进入下一个循环，判断当前的`i`是否小于等于`100`，依次执行直到`i=101`，则跳出循环。
    - 如果`i`大于等于`100`不是则跳出循环。
  - 最后通过`printf`函数打印出结果。
- ### 语句注释 ###
- ``int i, sum =0 ;``表示初始化`i`和`sum`为整型数0；
- ``i++;``等价于`i = i+1;`,`=`表示赋值号而不是等号,即将当前的i加1赋给新的i；
- ``sum+=i;``等价于`sum = sum + i`；即将当前的`sum`加上当前的`i`赋给新的sum。
- ``printf("%d\n",sum);``是一个标准输出库函数，其定义在`stdio.h`这个头文件中。
